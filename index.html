How to Break The Youtube
Open Google Chrome and drag-and-drop this link to your bookmark bar:

Break The YouTube

Then, go to a YouTube music video, like this one for example, and click on the bookmark after the YouTube page has fully loaded.

macOS users
If you can't drag-and-drop the link, right-click the link and 'Copy Link Address'. Then, bookmark this page, click 'Other...', rename the bookmark to something like 'BreakTheYoutube', clear the URL field and and paste the link you just copied instead.

Keyboard controls:
Q/W/E/A/S/D: Toggle effects.

Warning:
This project was only tested with Google Chrome and the current YouTube version served to my computer. It might not work on all YouTube versions.
